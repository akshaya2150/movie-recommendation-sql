# 🎬 Movie Recommendation & Rating Analysis System

This project is a SQL-based system designed to store and analyze movie data, user ratings, and watch history to generate useful insights and recommendations.

---

## 🚀 Features

* Manage users, movies, ratings, and watch history
* Identify top-rated movies using average ratings
* Find most popular genres
* Detect trending movies based on recent watch activity
* Analyze user behavior (movies watched)
* Recommend movies based on similar user preferences

---

## 🛠 Technologies Used

* MySQL
* SQL (Joins, Aggregations, Subqueries, Window Functions)

---

## 📂 Project Files

* `schema.sql` → Creates database and tables
* `data.sql` → Inserts sample data
* `queries.sql` → Contains analysis and recommendation queries

---

## ⚙️ How to Run

1. Open MySQL Workbench
2. Run `schema.sql` to create database and tables
3. Run `data.sql` to insert data
4. Run queries from `queries.sql`
